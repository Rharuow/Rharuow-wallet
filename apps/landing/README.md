# landing

Landing page promocional do RharouWallet em Next.js.

## Scripts

```bash
npm run dev --workspace=apps/landing
npm run build --workspace=apps/landing
```

## Deploy

Criar um projeto separado na Vercel apontando o Root Directory para `apps/landing`.

## Variaveis de ambiente

- `WALLET_URL`: URL da aplicacao principal para CTAs de login, cadastro e demais links da landing.
